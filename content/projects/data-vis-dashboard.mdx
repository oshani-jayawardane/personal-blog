## Overview

The post is not complete or organized. this is a rough skeleton

Dataset is curated by ourselves through web scraping from https://aviation-safety.net/. Selenium was used for scraping.
NLP for data extractions using tokens: using mistralai/Mistral-7B-Instruct-v0.2
Geopy to locate coordinate location of accidents based on latitude and longitude information 
IATA airport data for coordinates

Data cleaning
Schneiderman's mantra
- overview first
- zoom and filter
- details on demand

Gestalts Laws
- proximity
- similarity
- symmetry
- continuation
- closure
- figure-ground
- common featured

Tufte's design principles
- data ink ratio
- minimizing chart junk
- lie factor
- using small multiples
- layering and seperation
- high data density
